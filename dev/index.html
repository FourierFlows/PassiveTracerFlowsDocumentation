PassiveTracerFlows.jl Documentation

Overview

PassiveTracerFlows.jl is a collection of modules which leverage the FourierFlows.jl framework to solve for advection-diffusion problems on periodic domains.

Unicode

Oftentimes unicode symbols are used in modules for certain variables or parameters. For example, κ is commonly used to denote the diffusivity, or ∂ is used to denote partial differentiation. Unicode symbols can be entered in the Julia REPL by typing, e.g., \kappa or \partial followed by the tab key.

Read more about Unicode symbols in the Julia Documentation.
